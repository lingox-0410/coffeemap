# ☕ CoffeeMap

> 一个 Apple 风格、简约高级的**咖啡风味鉴赏记录工具**。
> 把每一杯咖啡的产地、豆种、处理法、风味与评分，沉淀成你的私人风味地图。

白底 · 留白 · 暖棕强调色 · 流畅动效 · 数据本地化（隐私无忧）· **零构建、可离线**。

---

## 运行方式

**方式一：本地服务器（推荐）**
```bash
cd CoffeeMap
python3 -m http.server 4178
# 浏览器打开 http://localhost:4178
```

**方式二：直接打开**
双击 `index.html` 即可（依赖已全部本地化，无需联网）。

首次打开会自动填充 11 条示例记录，方便你立刻体验。底部「清空」可重置为你自己的数据。

## 功能总览

### 录入（标签框 + 图示）
咖啡信息（国家/产区 · 庄园/地块 · 子产区 · 豆种 · 处理法 · 烘焙度 · 海拔 · 风味）
+ 鉴赏信息（咖啡店 · 鉴赏时间 · 冲煮方法 · 5 分制打分）+ 价格 · 笔记 · 照片（拖拽上传）。
每个标签框都带搜索建议、emoji 图示，并支持自定义。

### 输出
- **明细表** — 全字段底层数据，点表头排序，可搜索/筛选。
- **统计** — 数量 & 评分多维仪表盘 + 风味 DNA 雷达 + 13 类风味热力图。
- **世界地图** — 按数量/评分着色，点击国家或圆点下钻产地知识与记录。
- **明细卡片** — 每条记录一张卡，**所有标签可点击 → 跳转知识卡片**（豆种/风味/处理法介绍）。

### 富交互
🔍 全局搜索 · 🖼️ 生成分享图（PNG 下载）· 🔥 风味热力图 · 📷 照片相册自动汇总。

### 发散玩法
🛂 咖啡护照（集章 + 成就）· 🧬 风味 DNA 雷达 · ✨ 年度报告 · 🎯 盲品猜产地。

## 数据如何保存？怎么分享给朋友？

**数据保存在哪**
- 你录入的所有记录（含照片）都存在**当前浏览器的 `localStorage`**（键名 `coffeemap.records.v1`），**不上传任何服务器**，纯本地、隐私无忧。
- 因此数据是「**跟着这台设备+这个浏览器**」的：换电脑、换浏览器、清缓存都不会自动同步。
- 照片以压缩后的 base64 存入本地，`localStorage` 上限约 5–10MB，照片很多时建议定期导出备份。

**备份 / 迁移到另一台设备**
- 底部「**导出数据**」→ 得到 `coffeemap-data.json`；在新设备打开工具后「**导入数据**」选这个文件即可（会提示覆盖，导入前可先导出当前数据备份）。

**把「工具」分享给朋友的三种方式**
1. **发压缩包**：把整个 `CoffeeMap` 文件夹打包发给朋友，对方解压后双击 `index.html` 即可用（依赖已本地化，**可离线**）。朋友看到的是空白工具+示例数据，记录的是他**自己**的数据。
2. **免费托管成网址**（推荐，便于多人各自使用）：把文件夹拖到 [Netlify Drop](https://app.netlify.com/drop) / Vercel / GitHub Pages，得到一个链接发出去。每个访问者的数据都存在各自浏览器里，互不影响。
3. **分享单条记录**：在记录明细里点「🖼️ 生成分享图」，导出一张精美 PNG 发朋友圈/群——这是把「内容」而非「工具」分享出去。

> ⚠️ 工具本身没有云端账号体系，不会在用户之间同步数据。要和朋友共用同一份记录，目前用「导出 JSON → 对方导入」的方式手动合并。
