# CoffeeMap · 咖啡风味鉴赏记录工具 · 产品设计文档（PRD）

> 角色：产品经理 → 项目经理
> 一句话定位：**用 Apple 般的简约高级，把每一杯咖啡的风味与产地，沉淀成你的私人风味地图。**

---

## 1. 目标用户与价值

| 用户 | 痛点 | CoffeeMap 提供 |
|---|---|---|
| 精品咖啡爱好者 | 喝过很多豆子，记不住、没沉淀 | 结构化记录 + 风味地图 + 知识卡片 |
| 咖啡学习者 | 产区/豆种/处理法术语多，记不牢 | 每个标签可点开成知识卡 |
| 内容分享者 | 想晒、想复盘 | 一键分享图、年度报告、相册 |

设计原则：**白底、留白、克制配色（espresso 暖棕为唯一强调色）、大字号、流畅动效、即点即用、数据本地化（隐私）**。

---

## 2. 信息架构（6 大视图）

```
顶部导航：地图 · 卡片墙 · 明细表 · 统计 · 相册 · 护照   | 搜索 | ＋记录一杯
```

1. **地图** — 世界地图（等值着色）+ 产地下钻
2. **卡片墙** — 每条记录一张明细卡片
3. **明细表** — 底层全字段数据，可排序/筛选/搜索
4. **统计** — 多维数量 & 评分仪表盘 + 风味热力图 + 风味 DNA 雷达
5. **相册** — 所有记录照片自动汇总成瀑布流
6. **护照** — 产地集章 + 成就 + 年度报告 + 盲品游戏

---

## 3. 数据模型（一条记录）

| 字段 | 类型 | 输入方式 |
|---|---|---|
| 咖啡名称 | 文本 | 文本框 |
| 国家/产区 | 单选标签 | **标签框**（搜索+图示）|
| 庄园/地块 | 文本 | 文本框 |
| 子产区 | 多选标签 | **标签框**（建议+自定义）|
| 豆种 | 多选标签 | **标签框** |
| 处理法 | 多选标签 | **标签框**（带色点）|
| 烘焙度 | 单选标签 | **标签框** |
| 海拔 | 单选标签 | **标签框** |
| 风味 | 多选标签 | **标签框**（SCA 风味轮，13 类）|
| 咖啡店 | 文本 | 文本框 |
| 鉴赏时间 | 日期 | 日期选择 |
| 冲煮方法 | 单选标签 | **标签框** |
| 风味打分 | 0.5–5 ★ | 星级（支持半星）|
| 价格 | 数字 | 文本框（选填）|
| 品鉴笔记 | 文本 | 多行 |
| 照片 | 图片[] | 拖拽/点击上传（自动压缩）|

## 5. 输出层（需求对应）

- **底层明细表**：全字段表格，点表头排序，支持搜索/筛选，行内标签可点。
- **统计**：总杯数 / 产地数 / 均分 / 最爱产地；产地杯数榜、处理法&烘焙度环形图、冲煮榜、产地评分榜、风味 DNA 雷达、13 类风味热力图。
- **可视化**：
  - 世界地图按「数量 / 评分」切换着色深浅，点击国家或产地圆点 → 下钻该产地知识 + 记录卡片。
  - 每条记录一张明细卡片；卡片上所有信息标签可点击 → 知识卡片（豆种/风味/处理法等介绍）。

---

## 6. 风格与交互

- Apple 风：白底、毛玻璃导航、`-apple-system` 字体、大字弱色、`cubic-bezier` 动效、卡片轻投影。
- 富交互：全局搜索、生成分享图（PNG 下载）、风味热力图、照片相册汇总、拖拽上传、键盘可达的标签下拉。

---

## 7. 发散功能（额外 4 个，已实现）

1. **🛂 咖啡护照** — 世界产地集章 + 8 项成就系统（环球咖啡客 / 瑰夏猎人 / 厌氧实验家 …），游戏化收集。
2. **🧬 风味 DNA 雷达** — 聚合你全部记录的 13 类风味偏好，一眼看懂"你的味觉指纹"。
3. **✨ 年度报告 Wrapped** — Spotify-Wrapped 式年度总结（杯数 / 最爱产地 / 年度之杯 / 均分），可分享。
4. **🎯 盲品猜产地** — 隐藏产地、只给风味与处理线索，四选一竞猜 + 连胜计分，边玩边学。

> 可继续延展：冲煮配方复现、性价比追踪、口味相似推荐（已在数据层预留 price / aspectScores 字段）。

---

## 8. 技术与交付

- 纯前端、零构建：`HTML + CSS + 原生 JS`；依赖 D3 + TopoJSON + html2canvas **全部本地化，可离线**。
- 数据 `localStorage` 持久化，支持导出 JSON、清空、首次自动填充示例数据。
- 一键运行：`python3 -m http.server` 或直接打开 `index.html`。
